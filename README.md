# Virtual Piano

p5.js를 이용한 인터랙티브 가상 피아노 웹 애플리케이션   
마우스 클릭으로 피아노 건반을 연주할 수 있으며, 실시간 사운드 피드백과 시각적 효과를 제공합니다.

## 주요 기능

- 2옥타브 범위의 피아노 건반 (C4 ~ B5, 총 24건반)
- p5.js Oscillator를 이용한 실시간 사운드 생성
- 마우스 클릭으로 건반 연주

## 기술 스택

- p5.js - 그래픽 및 사운드 처리
- p5.sound.js - 오디오 신디사이저

## 사용 방법

1. 웹페이지가 로드되면 "아무 건반이나 클릭하여 피아노 연주를 시작하세요 🎈" 메시지가 표시됩니다
2. 피아노 건반을 마우스로 클릭하면 해당 음이 재생됩니다
3. 마우스를 떼면 소리가 멈춥니다
4. 검은 건반과 흰 건반 모두 클릭 가능합니다

## 커스터마이징

### 사운드 변경
```javascript
// sketch.js의 setup() 함수에서
oscillators[i] = new p5.Oscillator('sine');  // 'sine', 'triangle', 'square', 'sawtooth' 중 선택
```

### 볼륨 조절
```javascript
// playNote() 함수에서
oscillators[index].amp(0.4, 0.05);  // 첫 번째 숫자가 볼륨 (0.0 ~ 1.0)
```

### 캔버스 크기 변경
```javascript
// setup() 함수에서
createCanvas(800, 320);  // 너비, 높이
```
